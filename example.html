Example Google Chrome Extension

An example extension was created for Google Chrome to show off some of the features of Blabble. The extensions allows registration to a SIP server and the ability to make/receive calls. It's a good example for how to use the JavaScript API with the plugin. The code can be found here. The extension can be downloaded from here. It contains Blabble binaries for Windows, Linux, and OS X.

In later versions of Google Chrome the extension will need to be manually installed. First download the .crx file. Next, click the Chrome menu (the button with the three lines just left of the address bar) and select Tools -> Extensions. Now drag the .crx file into that window. Google Chrome will then ask you to confirm installing the extension.
